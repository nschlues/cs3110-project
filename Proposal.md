# Project Proposal: Free chess website
This is my proposal for my CS3110 project.

## What
I want to create a chess website like Chess.com or Lichess. Eventually, it would provide many of the game analysis, puzzles, lessons, streaming, chess variants and custom bots that these sites provide, but without charging premium membership for them. 

## Who
This site would target chess players who are average and do not want the high level competition of FIDE players like Magnus Carlsen and others like him. They just want a good chess site that lets them analyze their play and learn without paying for a subscription. These sites currently show adds and require a subscription for add free play and analysis. My website would be for players who do not want to pay for these services or put up with this. 

## How
I want to start with getting a working chess board that would let 2 players play on the same device. Next I would want to enable multiple players to plaay online together. After that, here are some features I have in mind:
- User profiles
- Game timer
- Account linking from Chess.com to analyze previous games played
- Rated puzzles
- Rating system
- Chess variants (chess960, bughouse, etc.)
- Synchronous games between users
- Asynchronous games between users (Daily or weekly)
- Bots with different levels of ability
- Game tracker with chess notation
- Pre-moving for synchronous games
- Conditional moves for asynchronous games
- Matchmaking for users based on rating ONLY
- Streaming of live tournaments or events

Here are some APIs I might use for this:
- Chess.com cloud analysis api
- Lichess Cloud analysis
- This can also be done locally
- Chess.com API
- Lichess API
- Firebase Auth API
